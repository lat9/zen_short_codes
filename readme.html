Introduction

This encapsulated plugin emulates the shortcode processing provided by WordPress, enabling a means to inject links to categories and products within your site's category- and product-descriptions, reducing the maintenance required for those links.

For additional information, refer to the plugin's GitHub Wiki articles.

History and Credits

The concept for parsing a shortcode's attributes is based on this WordPress documentation.

This readme's formatting based on that originally developed by @marco-pm.

Prerequisites

  • A Zen Cart installation, v1.5.8 or later.
  • PHP 8.0 or later.

If your site is using the Google Product Search Feed II, you'll need to use v1.0.4 or later for the shortcodes to be mapped to their respective links for the feed's generation!

Installation

The feature is released as a Zen Cart encapsulated plugin, so installation (and removal) is simplified.

  1. Merge the plugin's /zc_plugins directory with your site's pre-existing /zc_plugins directory.
  2. Log into your Zen Cart admin panel.
  3. Use the admin menus, navigating to Modules :: Plugin Manager.
  4. You'll see a plugin named Zen Cart ShortCodes; click the Install button for the plugin. That's it!

Usage

Once you've installed the plugin, you can enter shortcodes in your product- and category-descriptions. Those shortcodes are 'converted' into the appropriate links to the product or category when displayed on your site's storefront. Refer to the plugin's GitHub Wiki articles for additional information.

Uninstall

To uninstall the report, navigate to the admin's Modules :: Plugin Manager, find the line-item for Zen Cart ShortCodes and click the Un-Install button.
